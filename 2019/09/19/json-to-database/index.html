如何將Json存入資料庫當中

春雨
2019-09-19 (Updated: 2020-03-16)
程式 › SQL › 開坑
Database, SQL

有兩種解決的方法

  1. 在資料庫支援的情況下, 以Documention為輔完成任務
  2. 沒有資料庫不支援的情況下, 將Json的每個key拆開變成獨立的表格, 並加上ID建立所謂的關係來連結不同的表格

Table example:

table: Name

ID Name Hobby Age
1 Jane Cooking 10
2 Lucy Jogging 18

table: Pet

ID Name species Age
1 David Dog 4
2 Kitto Cat 4
